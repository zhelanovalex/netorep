Curses lab
Задача 2.
Высоконагруженная база данных MySql, критичная к отказу, должна располагаться на физических серверах, так как более полно будут использоваться ресурсы без побочных нагрузок,
с кластеризацией на уровне OS - pacemaker + DRBD или MySQL Replication.
Различные web-приложения - располагаются как правило на однородной ОС и могут раполагаться на виртуализации уровня ОС - docker.
Windows-системы для использования бухгалтерским отделом; - это паравиртуализация XEN в режиме PV, так как для бухгалтерии требуется стабильная работы систем по производительности.
Системы, выполняющие высокопроизводительные расчёты на GPU, должны располагаться на множестве отдельных физических специализированных серверов для параллельной работы.
Задача 4.
Проблемы и недостатки гетерогенной среды виртуализации - в первую очередь это обслуживание, мониторинг и диагностика неисправностей. Так же сервисное сопровождение "разного" обрудования. Еще есть серьезные вопросы 
по лицензированию. Необходимо выбрать универсальную систему виртуализации и переводить прилодения на нее. Если есть выбор то надо создавать единую систему виртуализации.
